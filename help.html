Pure Translator Help Guide

Getting Started

Pure Translator is a browser extension that allows you to translate text and web pages without relying on external translation services. All translations happen locally in your browser.

Basic Usage

  1. Select text on any webpage
  2. Click the floating translate button that appears
  3. View the translation in the popup

Alternatively, you can right-click on selected text and choose "Translate" from the context menu.

Features

Quick Translation

Translate selected text instantly with the floating button or context menu.

Page Translation

Translate entire web pages with a single click. You can restore the original page at any time.

Translation History

View and manage your past translations. You can export your history as a JSON file.

Custom Dictionary

Add your own translations to improve the translation engine.

Keyboard Shortcuts

  • Ctrl+Shift+T - Open page translation controls
  • Escape - Close all translation popups

Troubleshooting

Translations not working

If translations aren't working properly:

  1. Refresh the page
  2. Make sure you're not trying to translate very long texts (limit is 5000 characters)
  3. Check that the selected languages are correct

Page translation issues

If page translation doesn't work:

  1. Refresh the page first
  2. Try translating smaller sections of the page
  3. Some complex pages may not translate perfectly
